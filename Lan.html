HTML

HTML is the foundation of web development. It structures content on the web, providing the backbone for your pages. With HTML, you define the structure of your document using tags like <div>, <p>, and <h1>.

CSS

CSS adds style to your HTML structure. It allows you to control the layout, design, and visual presentation of your web pages. In this project, we're applying CSS to create a visually appealing and user-friendly design with proper alignments, colors, and more.

JAVA SCRIPT

JavaScript adds interactivity and dynamic behavior to your web pages. While this landing page primarily focuses on HTML and CSS, understanding the basics of JavaScript is crucial for creating interactive and engaging websites.
